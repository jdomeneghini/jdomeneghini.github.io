Prevendo valor de venda e aluguel de apartamentos - Machine Learning Web Application

Este projeto se trata de uma web application para prever valores de imóveis, usando um modelo de machine learning.

Este app foi desenvolvido usando Streamlit, uma biblioteca open-source fácil e prática para criação de apps e para o deployment deste app usei o Heroku, uma plataforma em nuvem onde te permite implantar, gerenciar e dimensionar apps em várias linguagens de programação.

Para acessar a aplicação deste projeto acesse o site: https://predictrealstateprice.herokuapp.com

Inicialmente foi feita uma analise exploratória dos dados afim de observar como cada atributo influenciava no valor do imóvel. Abaixo alguma informações observadas na análise.
